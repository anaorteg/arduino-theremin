void loop ( )

The standard Arduino loop function.

This is the normal Arduino loop function where the main code execution in Arduino software normally occurs. In the case of this software, the loop is left empty as the work is all accomplished in timer interrupts.

void setup ( )

The standard Arduino setup function.

The setup function for this program does the following:

  • Configure the Arduino General Purpose I/O pins for INPUT and OUTPUT.
  • Configure the ATmega Timer/Counters and Watchdog Timer.

void timer1_setup ( )

Set-up Timer/Counter 1 registers.

Configures the Timer/Counter 1 registers so that the rising edges of the incoming frequency is counted. The counter is too small, even at 16-bits, to count all of the edges in the time alloted so the timer is configured to interrupt on overflow so we can also keep track of how many times it overflows.

void timer2_setup ( )

Set-up Timer/Counter 2 registers.

void watchdog_setup ( )

Set-up the Watchdog timer registers.

Configures registers to run the Watchdog timer to interrupt at an interval of 16 ms. Also ensures that the timeout of the watchdog will not trigger a reset of the ATmega chip.
